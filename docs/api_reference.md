# BankSaathi API Documentation

## Webhooks
### POST `/api/v1/webhook`
Handles WhatsApp Business API messaging webhook payloads (audio transcripts and text).

### GET `/api/v1/webhook`
Verifies webhook registration handshake with token validation.

## Admin API
### GET `/api/v1/admin/metrics`
Exposes system-wide operational metrics for sessions and DMS transitions.
